cmpctircd
=========

About
-----
The aim of this project is to provide a stable, fast, and *modern* ircd.

Status
-----
Under heavy development. It'll be clear when it is production ready. Feel free to report any bugs on [bugzilla](https://bugs.cmpct.info/).
